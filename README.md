## üìñ Tentang Project

Project **Knowledge-Test** adalah sistem web sederhana yang saya buat sebagai latihan untuk mengembangkan aplikasi fullstack menggunakan **React JS** (Frontend) dan **Node.js + Express** (Backend) dengan database **MongoDB**.

Fokus project ini adalah:

- Menampilkan data produk dari backend
- Menampilkan gambar produk dari endpoint `/api/products/top`
- Membuat Hero section dengan carousel dinamis
- Membuat Navbar yang fixed-top dan responsif
- Membuat sistem modal Login & Register
- Menghubungkan frontend & backend memakai REST API

---
